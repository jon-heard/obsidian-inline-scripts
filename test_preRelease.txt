1. Open Obsidian
2. Enable plugin, disable, then renable it

SHORTCUTS - RUNEXTERNAL FUNCTION !! NON-MOBILE ONLY !!

		- Expansion: return runExternal("start /wait timeout 5 && dir", false, true);

16. Type ";;test 8;"
